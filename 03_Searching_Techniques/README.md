# Searching Techniques

This folder contains basic searching algorithms implemented in a beginner-friendly way.

## Algorithms Included
1. Linear Search
2. Binary Search

## Description
Searching algorithms are used to find the position of an element in a given dataset.

Linear Search checks elements one by one.
Binary Search works on sorted data using divide and conquer method.

## Learning Outcome
- Understood searching logic
- Learned time complexity differences
- Practiced sorted vs unsorted search methods
